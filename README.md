# Cruxa
Cruxa: The ultimate social network and activity tracker for climbers.

## Документация Проекта (Docs)
Вся архитектура, планы и требования описаны в следующих документах:

1. [Product Requirements Document (PRD)](docs/PRD.md) — Идея, фичи MVP, целевая аудитория.
2. [Functional Requirements (FRD)](docs/FunctionalRequirements.md) — Детальное описание пользовательских сценариев (роли, логирование в зале, профили).
3. [Architecture & Tech Stack](docs/Architecture.md) — Выбранный стек технологий (C#, React, Flutter).
4. [Database Schema](docs/Database.md) — Сущности и организация данных (PostgreSQL).
5. [UI/UX & Design System](docs/Design.md) — Гайдлайны Material Design и цвета категорий.
6. [Roadmap](docs/Roadmap.md) — Пошаговый план разработки продукта соло-инженером.
7. [Workflow & Agreements](docs/Workflow.md) — Договоренности о процессах, ADR и обязательном тестировании.
